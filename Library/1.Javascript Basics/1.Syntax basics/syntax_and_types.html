String methods

pour faire une concatenation si c'est une phrase, rajouter l'espace vide entre les mots

utiliser la method .length pour avoir la longueur du mot

utiliser la method .toUpperCase pour avoir le mot en majuscule

utiliser la method .indexOf(argument) pour trouver la position de l'argument

  • .lastIndexOf(argument) pour avoir la dernière position de l'argument
  • .slice(arg1, arg2) sur une variable pour afficher entre deux points
      si arg1 = arg2 renvoi string vide
      si arg1 est supérieur à arg2, renvoi une string vide
      si arg2 est oublié, affiche tout à partir de arg1
      si les arg sont supérieurs à la longueur de la string, renvoi une string vide
  • .substring(arg1, arg2) sur une variable pour afficher entre les deux points
      si arg1 = arg2 renvoi string vide
      si arg1 est supérieur à arg2, la méthode inverse les deux valeurs
      arg1 est le point de départ, arg2 la longeur prise, et non la position, ex (2,10), début à 2 et fin à 12. Si arg2 > à length, alors on arrête à la fin de la string
  • .replace pour remplacer la première lettre ciblée par une autre ('arg1', 'arg2'), arg1 est l'arguement remplacé par arg2

pour intégrer une variable dans une string, utiliser le `` pour créer la string, et appeler les variables avec ${variable}

on peut créer des méthodes js, avec des tags HTML pour les afficher dans la console

Number methods

operateurs mathématiques: +, -, *, **( pour la puissance), /, %

ordre de calcul dans une opération:

      Parenthèses
      Puissances
      Division
      Multiplication
      Addition/Soustraction

utiliser le ++ pour rajouter +1, ou -- pour faire -1

+= arg pour additionner la variable avec arg, -= arg, pour soustraire, /= arg pour diviser la variable par arg, et *= pour mutiplier

pas possible de faire des opérations entre des nombres et des strings

Array methods

utiliser les [], pour récupérer juste un des éléments de l'array

ex: famille[1]

.length pour avoir le nombre d'élements de l'array

.join sur un array permet d'insérer tous les éléments dans une string,

.concat permet d'insérer une array dans une array

.push permet d'ajouter un élément à une array, et retourne le nombre d'éléments de l'array

.pop permet de détruire le dernier élément d'une array et retourne cet élément

Null et Undefined

si une variable n'est pas définie == Undefined

Boolean methods

.includes permet de retourner un boolean si un élément est présent. ça peut être une string ou un nombre

utiliser un = pour assigner à une variable, et == pour faire une comparaison, et === pour faire une stricte égalité

la comparaison retourne false si une string n'est pas strictement identique à l'élément comparé

on peut utiliser les comparaisons entre strings, pour comparer les lettres de l'alphabet, les lettres les plus loin étant > aux lettres plus proche du début

ex: z > a

les lettres majuscules sont aussi > aux lettres minuscules

JS considère qu'une string est égale à un nombre, car au moment de la comparaison, il fait une conversion

Donc si le nombre en tant que nombre, est égal à un nombre dans une string, il retourne true

une stricte comparaison rendra un false

Types conversions

On peut changer le type d'une variable dans JS indiquant avant la conversion son nouveau type

on peut utiliser la méthode typeof pour vérifier le type d'une variable

on peut pas changer un mot en nombre, renvoi NaN

on peut changer un nombre en String

on peut changer un élement en Boolean, un nombre positif ou négatif, ex: 100, renvoi true, mais 0 renvoi false

une string est également un Boolean qui renvoi true
