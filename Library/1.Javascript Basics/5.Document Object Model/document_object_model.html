le document créé dans le navigateur est modelé par l'objet document créé par le navigateur

dans le nav, c'est le "document object", et ce document contient par défaut un grand nombre de méthodes disponibles

pour cibler avec du JS un élément dans une autre page, il faut créer une const dans JS, et faire un query selector. cela cible le premier élément de la page

pour cibler une classe, il faut nommer la classe dans la variable, ex : ('.class') ou (#ID).

on peut autrement faire un getElementById pour cibler un ID,

si on a deux fois la même classe, mais que l'on veut n'en cibler qu'une, il faut spécifier l'élément dans le querySelector

on peut aussi directement cibler dans la console l'élément que l'on veut récupérer, et copier sa position pour la placer dans le query selector

pour cibler plusieurs éléments identiques, querySelectorAll, cela vaut pour les classes et élements

pour cibler tous les éléments identiques, par exemple les p, dans une page HTML, utiliser getElementByTagName, cela crée une collection de tous les éléments ciblés

Ajouter, modifier et supprimer du contenu

on peut modifier du texte inséré dans le HTML dans la page JS, en ciblant l'élément que l'on veut modifier.

on peut alors utiliser .innerText sur l'élément pour le modifier

pour modifier plusieurs éléments identiques, faire un querySelectorAll des éléments voulus, et créer la méthode avec un forEach

on peut aussi modifier tout le contenu HTML, avec .innerHTML . En faisant ça, on peut modifier le tag ( ex: un p en h1), ainsi que le text

pour modifier le lien vers un site, on fait un querySelector du a

le a à un attribut qui est le href. pour modifier l'attribut, on utilise la méthode setAttribute sur le href, en passant deux arguments

le premier arguement est l'élément ciblé, donc le href, et le deuxième argument est le nouvel attribut

ex: link.setAttrbiute('href', 'https:www.google.com');

on peut ajouter une classe à un tag, en le ciblant, et en passant les deux arguments avec la méthode setAttribute

on peut aussi changer son style. cela change complètement le style qui existait avant

pour ajouter des modifications de style en complément de ce qui existe déjà, on crée une variable avec l'élement ciblé, et on ajoute la méthode .style

.style permet de modifier les propriétés de l'élement ciblé, comme la marge, padding, couleur, font-size, sans overwritter ce qui a déjà été fait

pour supprimer une propriété, ciblé la propriété voulue, et lui mettre une valeur égale à une string nulle

pour ajouter une classe à un élément qui n'en a pas un, on crée une variable et on cible l'élément voulu avec un querySelector, on ajoute la propriété .classList et .add

cela permet de rajouter la classe

pour supprimer une classe, même syntaxe, mais au lieu de add, on utilise .remove
