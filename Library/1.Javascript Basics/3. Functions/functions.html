Functions

Détails sur les fonctions

Opérateur function

fonction déclarative

pour créer une fonction déclarative, il faut la déclarer avec function, puis la nommer

on ajoute le code ensuite, puis on peut l'appeler à n'importe quel moment, même en amont de la fonction

fonction expressive

une fonction expressive est une fonction créée avec une varaible constante, puis un nom, suivi du code

une fonction expressive ne peut être appelé qu'une fois qu'elle a été créée, donc sous la fonction

ajouter des paramètres aux fonctions

on peut ajouter des paramètres aux fonctions, comme par exemple un nom quand on appelle quelqu'un

cependant pour retourner le paramtère, on ne peut le faire que dans la fonction, car c'est une variable locale

pour passer plusieurs paramètres, on les ajoute à la suite dans la fonction

on peut insérer des valeurs par défaut aux paramètres, en les intégrant directement dans la fonction

retourner les valeurs

pour retourner les valeurs sans faire un console.log, on écrit la fonction, et on insère un return

on peut alors appeler le résultat en dehors de la fonction

Fonctions fléchées (arrow function)

Lien mozilla

une arrow function fonctionne comme une fonction régulière, mais est plus rapide à écrire

si on a un seul paramètre, on peut supprimer les parenthèses autour

aussi, si on a un seul retour dans la fonction, on peut simplifier en suprrimant le retour et les accolades, et en mettant tout sur une seule ligne

si on a aucun paramètre, (ici raidus), il faut quand même mettre les parenthèses vides

callbacks functions

une callback fonction est une fonction passée dans une autre fonction en tant qu'argument, qui est ensuite invoquée à l'intérieur de la fonction externe pour accomplir une action

Méthodes

il y a peu de différences entre une méthode et une fonction.

une méthode est comme une fonction qui est associée à un objet, ou un objet de la base de donnée
