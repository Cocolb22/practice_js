XMLHttpRequest

il faut faire une new XMLHttpRequest(). cela fonctionne pour tous les types de données, XML, JSON, text brut...

on ouvre ensuite la variable avec open, et on passe deux arguements. Le premier est le type de requete, GET, POST, PUT...

Voir tous les types de requetes

le deuxième argument est le end-point, l'endroit ou on veut récupérer les données, c'est donc l'adresse du lien

il faut ensuite envoyer la requete avec la méthode send

pour savoir quand on a eu une réponse, il faut créer un eventListener qui va permettre de savoir le stade où en est la requete
